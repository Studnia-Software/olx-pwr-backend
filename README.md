# olx-pwr-backend

## Setup:
0. Install docker -> https://docs.docker.com/desktop/install/windows-install/
1. ```cd olx-pwr-backend```
2. ```cd docker```
3. ```docker compose up -d ```
4. ```create database olx-pwr-backend```
5. Power up app

 ## Testing:
 Go into `http://localhost:8080/swagger-ui` and enjoy
